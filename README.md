# AI-First CRM – HCP Module

## Overview
This project demonstrates an AI-powered CRM system designed for healthcare professionals (HCPs). 
The system logs interactions, analyzes sentiment, checks compliance, and recommends next best actions using **LangGraph + Groq LLM**.

## Features
- Log and edit HCP interactions
- Multi-step agentic workflow with 5 tools
- Groq LLM powered summarization and sentiment
- Compliance checks for life sciences
- Next best action suggestions for field reps
